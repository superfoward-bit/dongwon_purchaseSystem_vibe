## A. 구매 핵심 (pro)

### A-1. 구매요청 (PR, Purchase Request) ◆
- 물품 구매요청 등록/수정/삭제
- 용역(서비스) 구매요청 등록/수정/삭제
- 구매요청 품목 다건 관리 (수량/규격/희망납기/요청부서)
- 구매요청 상신·결재, 수정결재, 반려 후 재작성
- 구매요청 → 소싱(견적/역경매)/발주 전환
- 진행상태 관리: 작성요청→집수대기→결재대기→결재완료→경쟁요청→협력사선정→계약예정→계약완료
- 신규품목 동시요청(품목코드 채번 연계)
- 구매요청 진행현황 조회/추적

### A-2. 견적 (RFx = RFQ/RFP) ◆ (▲DFS Factory/Strategy 고도화)
- 견적요청(RFQ)/제안요청(RFP) 생성 (PR 기반)
- 협력사 초대·견적 공고
- 협력사별 견적가/제안서 입력 (협력사 포털)
- 가격 평가(최저가 등) / 비가격 점수 평가
- 평가표·평가위원 관리, 의사결정
- 선정/미선정/유찰/재견적/취소 처리
- 견적 결과 → 발주(PO) 생성
- 견적 결과 → 단가계약 생성
- 견적 유형별 전략 처리(RFQ/RFP/SAL) ▲
- 자동 공고시작/마감 스케줄러
- 진행상태: 작성→공고중→진행중→평가중→평가종료→종료(유찰/선정)

### A-3. 정보요청 (RFI) ◆
- 협력사 정보요청 등록/항목 구성
- 발송일·마감일 일정 관리
- 협력사 응답 수집 (포털)
- 자동 발송/마감 스케줄러
- 결과 검토

### A-4. 역경매 (Auction) ◆
- 역경매 공고 생성 (PR 기반)
- 협력사 입찰가 입력, 실시간 순위 산정
- 입찰 진행현황 모니터링
- 낙찰자 선정
- 역경매 결과 → 발주(PO) 생성
- 역경매 결과 → 단가계약 생성
- 자동 진행/마감 스케줄러
- 역경매 결재

### A-5. 입찰고정가 (Bidfix) ◆
- 역경매 종료 후 입찰가 고정 설정
- 입찰고정가 변경예약 (배치)
- 변경 결재
- SAP 연동 변경 반영
- 상태: 변경예약→완료/실패

### A-6. 단가계약 (Contract) ◆
- 단가계약 등록/수정/삭제 (헤더+품목)
- 계약 유효기간·금액(공급가/VAT/합계) 관리
- 협력사별/플랜트별 계약, 플랜트 확대
- 계약 첨부파일, 외부참조(전자계약) 링크
- 계약 결재, 수정결재, 일괄결재
- 계약정보 SAP 연동

### A-7. 발주 (PO, Purchase Order) ◆
- 발주 등록/수정/삭제
- PR 기반 / 견적결과 기반 / 역경매결과 기반 발주 생성
- 발주서 정보(PO번호·수정판수·협력사·납기·결제조건·통화)
- 발주 품목별 금액/수량 관리
- 발주 상신·결재, 수정결재
- 발주서 출력(OZ)
- 상태: 발주완료→입고요청→검수진행→검수완료/취소

### A-8. 입고/검수 (GR, Goods Receipt) ◆
- 입고(검수) 헤더/품목 등록
- 발주 기반 입고 등록, 검수 수량/가격/상태
- 검수 완료 처리, 입고 결재
- 입고변경(GrChg) 예약·결재
- 선급금/기성 청구(PaymentBill)
- 보증보험료(GrGrt) 관리
- 입고/구매 통합조회(GrPur)

### A-9. 품목단가 (ItemPrice) ◆
- 전용품목 단가 등록/수정/삭제
- 단가 변경 임시저장→결재→확정
- 단가 이력 관리
- 농산 일일단가 관리 ▽(농산 특화)
- SAP(INDIGO) 단가 동기화
- 상태: 변경→임시→결재반영→결재사인→IF설비→SAP전송→적용완료

### A-10. 인도가/운반비 (TransPrice) ◆
- 인도가(운반비) 변경예약
- 인도가 기본정보 입력
- 변경 결재
- 월별 인도가 계약
- SAP 연동 송신
- 상태: 임시→반영→신청→SAP전송→완료/실패

### A-11. 정산/마감 (CLS, Closing) ◆
- 월별/기간별 구매정산
- 마감금액 생성/현황/LOCK·UNLOCK
- 마감전표(서류) 생성
- 마감 입고(ClsGr)/마감 구매(ClsGrPur)
- 청구서 마감(Bill), 인보이스 마감(Inv)
- 선급금 마감(ClsAdvpmt)
- 입고변경 마감(GrChg), 보증보험료 마감(GrGrt)
- 갭정산(매입갭/분배갭/금액갭/매출갭)
- 정산결과 SAP 송신

### A-12. 시황 (Garak, 농산물 시황) ▽
- 농산물 일일 시황 수신 (평균가/전일대비/전년대비)
- 품목·등급별 시황 데이터 관리
- 단가 참고정보 제공

### A-13. 가격요소 (PriceFactor) ◆
- 가격요소 정의(기본/혼합)
- 가격군 관리, 가격군별 요소 매핑
- 견적/역경매 적용

### A-14. 실적 (PerformRes) ▲
- 마감실적 현황 조회/관리
- 납기실적 추적
- 구매담당자 메모

### A-15. 판매발주 (SalesPO) ◆
- 판매 관점 발주 정보 관리
- 고객/플랜트별 집계, 승인(Cst/Plt)

### A-16. 채권 (AR) ◆
- 선급금·보증금 등 미회수채권 조회
- 채권 회수현황 추적

## J. 공통 기반 / 비기능 (common, security, infra) ◆
- 멀티테넌시(SYS_ID 데이터 격리)
- 다국어(Locale) 전체 지원
- 첨부파일 업로드/다운로드
- XSS/CSRF 보안, 세션 관리
- 예외/에러 표준 처리, 로깅
- 문서번호 채번 공통
- 배치/스케줄러(Quartz): 견적/RFI/역경매/결재/협력사/변경예약/정산 등
- 협력사 포털: 협력사가 RFx·RFI 응찰, 정보수정, 전자계약 서명 직접 처리
- **변경이력 감사로그(RUD 추적) (신규)**: 주요 데이터의 생성/수정/삭제 추적 — 변경 전/후 값, 변경자, 변경일시, 변경사유 기록. 공통코드 등 기준정보 변경이력 포함. 테이블 단위 on/off 설정, 이력 조회/검색 화면 제공
- **문서 되돌리기 / 상태전이 표준 (신규·전 도메인 공통)**: 모든 프로세스에서 문제 발생 시 **이전 단계로 되돌리기** 지원 — 반려(REJECT)·반송(RETURN)·회수(RECALL)·취소(CANCEL)·재상정(REOPEN)·관리자 강제이동(FORCE). 상태전이 규칙은 `CM_STATUS_FLOW`로 데이터 관리(하드코딩 금지), 정/역방향 전부 `LOG_DOC_STATUS_HIS`에 이력 기록(사유 필수), 현재 상태별 가능한 액션 버튼 동적 노출.
